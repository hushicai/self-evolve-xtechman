## 测试执行方法

### 微信开发者工具测试

1. **打开项目**
   - 微信开发者工具 → 导入项目
   - 选择 `self-evolve-xtechman` 目录
   - AppID 使用测试号或 "touristappid"

2. **切换机型**
   - 工具顶部 → "机型" 按钮
   - 选择不同设备模拟器

3. **清除缓存测试**
   - 工具 → 清缓存 → 清除数据缓存
   - 验证数据持久化功能

4. **真机预览**
   - 点击"预览"按钮
   - 扫码在真机上测试

## 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 日历显示空白 | dateUtil 未正确导入 | 检查 require 路径 |
| 事件不显示 | events 数据格式错误 | 检查 events 数组结构 |
| picker 不弹出 | picker 组件配置错误 | 检查 picker mode 和 range |
| 滑动不响应 | touch 事件未绑定 | 检查 bindtouchstart/touchend |
