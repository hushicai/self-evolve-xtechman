**正式发布需要**：
1. 登录微信公众平台
2. 注册小程序账号
3. 获取正式 AppID
4. 更新 `project.config.json` 中的 appid 字段

```json
// project.config.json
{
  "appid": "wxXXXXXXXXXXXX"  // 替换为正式 AppID
}
```

### 4. 隐私协议

小程序需要配置隐私协议，说明数据收集和使用情况。

**XTechMan 隐私协议要点**：
- 仅存储用户事件数据（本地存储）
- 不收集用户个人信息
- 不进行数据上传或云端存储
- 数据仅用于日历事件管理功能
- 用户可随时清除数据

## 发布流程

### 步骤 1: 注册小程序账号

1. 访问 https://mp.weixin.qq.com
2. 点击"立即注册" → 选择"小程序"
3. 填写账号信息
4. 完成邮箱验证
5. 信息登记（主体类型选择）

### 步骤 2: 配置小程序信息

1. 登录小程序后台
2. 设置 → 基本设置 → 小程序信息
3. 填写名称、简介、描述
4. 上传头像图标
5. 选择服务类目

### 步骤 3: 获取 AppID

1. 开发管理 → 开发设置
2. 复制 AppID
3. 更新项目配置

### 步骤 4: 上传代码

1. 微信开发者工具打开项目
2. 点击"上传"按钮
3. 填写版本号和备注
4. 等待上传完成

### 步骤 5: 提交审核

1. 小程序后台 → 版本管理
2. 开发版本 → 提交审核
3. 填写审核信息
4. 等待审核结果（通常 1-7 天）

### 步骤 6: 发布上线

1. 审核通过后
2. 点击"发布"按钮
3. 小程序正式上线

---

## 审核注意事项

### 可能被拒绝的原因

| 原因 | 预防措施 |
|------|----------|
| 功能不完整 | 确保所有功能可用 |
| 界面不符合规范 | 遵循 WeUI 设计风格 |
| 隐私协议缺失 | 配置完整隐私协议 |
| 服务类目不符 | 选择正确类目 |
| 测试内容 | 移除测试数据/占位符 |

### 审核加速建议

- 功能完整、无明显 bug
- 界面美观、符合规范
- 提供测试账号（如需要登录）
- 清晰的功能说明
