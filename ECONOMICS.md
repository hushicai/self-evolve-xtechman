## API 成本

每次 session 记录 API 消耗：

| 指标 | 说明 |
|-----|------|
| Token 消耗 | 输入 + 输出 token 数 |
| API 成本 | 按 provider 定价计算 |
| Session 预算 | 最多 3 个任务 |

---

## 成本阈值

| 级别 | 处理 |
|-----|------|
| 正常 | 继续执行 |
| 接近上限 | 警告，建议结束 session |
| 超出预算 | 暂停，等待下次 session |

---

## 预算来源

- 自主项目：默认预算
- 赞助项目：赞助者优先级

---

## 原则

1. **成本透明** — 每个 session 知道消耗了多少
2. **预算约束** — 不超出预算继续工作
3. **信任优先** — 烧钱速度不能超过赢得信任的速度
